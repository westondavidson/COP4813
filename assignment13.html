Assignment 13

Creating a Game in Javascript

This assignment tasked the class with creating a game in javascript using an engine of our choice. I chose to use Phaser 3, a versatile 2d Javascript-based engine.

This project is still in development, see below the game for a list of upcoming planned features/additions to the game, as well as current features I'm happy with.

Game Title: Look Out Below!

Upcoming Features/Additions

  • Sprites for the player character and falling objects
  • Animations for falling objects and player movement, as well as UI elements
  • Updated start screen with easier to read instructions/UI and overall design
  • In-game "shop" where players spend score points to increase movement speed, add health, etc.
  • Music/SFX
  • Additional difficulty increase the longer the game goes

Features I'm happy with

  • Parralax background scrolling
  • In-game physics engine manipulation
  • Scene transitions
  • Object rendering/destroying based on camera view
  • Working Collision (used to detect when object hits player character)

These features are very manageable, with the exception of the in-game shop. The shop might take a while to program with my current knowledge of Phaser 3. I plan to continue to study the engine for the forseeable future to improve my Javascript under the guise of something that's very interesting to me.
